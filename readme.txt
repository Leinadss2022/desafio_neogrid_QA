# Testes de Automação com Cypress v14.0.0

# NodeJS e NPM instalados: https://nodejs.org/en/download

## Como rodar os testes

1. Clone o repositório:


   ```bash
   git clone <URL_DO_REPOSITORIO>

2. Instale as dependências:

npx install cypress


3. Execute os testes:

npx cypress open

Isso abrirá a interface gráfica do Cypress, onde você pode selecionar e rodar os testes no navegador de sua escolha.
